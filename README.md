# 🌤️ Weather App

A modern, responsive Weather Web Application built with HTML, CSS, and JavaScript. Fetch real-time weather information including temperature, weather conditions, humidity, and wind speed for any city.

## 🚀 Features

- 🔍 **Live City Search**: Search weather details for any city worldwide.
- 🌡️ **Temperature & Conditions**: Display temperature in °C along with visual weather icons.
- 💧 **Humidity & Wind Speed**: Track atmospheric details easily.
- 🎨 **Clean & Responsive UI**: Designed with smooth visuals and clear statistics.

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **OpenWeatherMap API**

## 💻 How to Run Locally

1. Clone this repository:
   ```bash
   git clone <YOUR_REPOSITORY_URL>
   ```
2. Open `index.html` in any modern web browser.
